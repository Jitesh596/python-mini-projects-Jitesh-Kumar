# Python Mini Projects
This repository contains two Python mini projects submitted as part of college coursework.

## 1. GradeBook Analyzer
Features:
- Manual entry of marks
- CSV file input
- Average, Median, Highest & Lowest calculation
- Grade assignment
- Pass/Fail analysis

## 2. Library Inventory Manager
Features:
- Add new books
- Issue and return books
- Search by title or ISBN
- Save and load data using JSON file
